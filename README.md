# hubot-info

A hubot script that does the things

See [`src/info.coffee`](src/info.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-info --save`

Then add **hubot-info** to your `external-scripts.json`:

```json
[
  "hubot-info"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
